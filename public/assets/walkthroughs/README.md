This directory contains assets for adventure game walkthroughs including:

- kings-quest-vi/ (King's Quest VI assets)

File naming convention:

- thumbnail.jpg (Main thumbnail for game listing)
- screenshot-001.jpg (Numbered screenshots)
- title-screen.jpg (Title screen capture)
- artwork-\*.jpg (Promotional artwork)

Image specifications:

- Thumbnails: 400x300px recommended
- Screenshots: Original resolution preserved
- Format: JPG for photos, PNG for UI elements
